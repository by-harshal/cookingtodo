# AI Cooking To-Do List

A lightweight, AI-driven micro-app that instantly converts your daily context into a practical, budget-aware meal plan with an actionable grocery list and smart substitutions.

## Core Value Proposition
Meal planning is often tedious and disconnected from our actual daily reality. This app takes your real-world constraints—your schedule, your budget, and your dietary preferences—and instantly generates a fully actionable plan using Generative AI. 

## Features
- **Context-Aware Planning**: Generates breakfast, lunch, and dinner based on your specific daily schedule.
- **Auto-Generated Grocery List**: Interactive checklist with AI-estimated prices.
- **Budget Feasibility**: Automatically checks if your meal plan fits your budget.
- **Smart Substitutions**: Suggests alternatives for expensive or hard-to-find ingredients.
- **Zero Backend**: Securely uses your local browser storage and direct API calls to keep the footprint extremely lightweight.

## Tech Stack
- **Frontend**: Vanilla HTML, CSS, JavaScript (Premium UI Shell)
- **AI**: Gemini 1.5 Flash API
- **Data**: Browser `LocalStorage`
- **Hosting**: GitHub Pages

## Setup Instructions
1. Open the [Live Link](https://by-harshal.github.io/cookingtodo/).
2. Enter your [Gemini API Key](https://aistudio.google.com/app/apikey). The key is stored securely in your browser's local storage.
3. Enter your day's schedule, budget, and diet.
4. Click **Generate Meal Plan** and let the AI do the heavy lifting.
